Work Samples

Several programming languages can be used for Machine Learning including Python, Java, R, C++ and more. Out of these, the most popular one is Python and that is also my first choice followed by R.
The framework I generally use is Google Colab, which is best in my understanding.
Following are short introductions to some of the important processes and model types in ML.

Feature Engineering

Biggest slice in the pie of total time that a data scientist spends on his/her model is undoubtedly feature engineering. The data we get is not always the way we want it to be. Many times it goes through processes of feature selection, feature extraction etc to make it as perfect as possible. Preprocessing is the most important pat of entire machine learning process.

Regression

Regression in ML terms is when we use many data samples to train a model that predicts a number for a new data sample, and this model is called a regression model of ML.

Classification

Classification in ML terms is when we use many data samples to train a model that predicts a category/class for a new data sample.

Clustering

Clustering is an ML application of unsupervised learning, where we use a vast dataset to explore those patterns in data, which were previously unknown.

Deep Learning

Deep learning uses neural networks, which are a series of algorithms that learn the patterns in a dataset through a process that mimics the operation of a human brain.
