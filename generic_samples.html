Machine Learning App Deployment

Miles Per Gallon Regressive Prediction

Visit the App Website

Data used is Auto MPG Dataset sourced from UCI Machine Learning Repository. This dataset was taken from the StatLib library which is maintained at Carnegie Mellon University. The dataset was used in the 1983 American Statistical Association Exposition.

Features include number of cylinders, engine displacement (cu. inches), horsepower, vehicle weight (lbs.), time to accelerate from O to 60 mph (sec.), model year (modulo 100), and origin of car (1. American, 2. European, 3. Japanese). The car model should be entered as a number less than 100 representing year 19(entered number), for exaomle: enter 87 for 1987. These features are used for training a regression model that predicts the Mile Per Gallon value for a given set of parameters representing a particular car model.

The predictive modelling included Pre-processing(Imputation, Standardization etc.), EDA, Data Preparation, Model Selection, Pipeline preparation, Hyperparameter tuning and Deployment. Flask and Heroku were used for Deployment of the model application.
